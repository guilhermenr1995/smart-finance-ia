Exemplos práticos para quem nunca exportou extrato. O caminho exato pode variar por versão do aplicativo.

Arraste para o lado para ver mais bancos

Nubank

  • Abra Conta ou Cartão.
  • Entre em Extrato.
  • Procure Exportar/Compartilhar (OFX para conta, CSV para fatura).

Itaú

  • No Internet Banking, vá em Conta Corrente.
  • Acesse Extrato no período desejado.
  • Use Exportar (geralmente OFX e, em alguns casos, CSV).

Bradesco

  • Acesse Extrato por período.
  • Toque em Exportar no rodapé/menu.
  • Priorize OFX para integração financeira.

Santander

  • Abra Conta Corrente.
  • Consulte o Extrato do período.
  • Use Exportar/Download (OFX/CSV conforme disponibilidade).

Banco do Brasil

  • Entre no menu de Extratos.
  • Selecione o período desejado.
  • Use Download/Exportação (OFX é comum).

Caixa e Inter

  • Busque por Extrato detalhado.
  • Abra o menu de compartilhar/exportar.
  • Quando não houver CSV, use OFX.
